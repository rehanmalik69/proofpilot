# ProofPilot, Explained Simply

## 1. Concept
ProofPilot is a website that helps someone organize proof when they have a problem with a company, like a refund, damaged order, bad service, or billing mistake. It lets them create a case, upload receipts or messages, and keep everything in one clean place.

## 2. GenAI Part
The app can read the case details and supporting files, then prepare a neat summary, timeline, missing-proof checklist, and complaint draft. If an AI service is available, it can use that; if not, it still works with a built-in local helper.

## 3. Stack
It is built as a modern web app using Next.js, TypeScript, and Tailwind CSS. Supabase handles accounts, database records, and uploaded files, while Vercel hosts the live website.
